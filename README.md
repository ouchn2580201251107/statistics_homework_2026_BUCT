# statistics_homework_2026_BUCT
北京化工大学继续教育学院 统计学课程作业(jupyter实现)

作业清单：作业均以.ipynb结尾，作业号与整个文件号相同，即：
- 统计学作业1.ipynb  ->  1.统计学的基本问题及统计资料整理
- 统计学作业2.ipynb  ->  2.统计指标分析
- 统计学作业3.ipynb  ->  3.区间估计
- 统计学作业4.ipynb  ->  4.假设检验
- 统计学作业5.ipynb  ->  5.一元回归分析
- 统计学作业6.ipynb  ->  6.时间数列分析
- 统计学作业7.ipynb  ->  7.统计指数分析

如在github上查看，可以直接看到代码和执行结果；如下载到本地，请用Jupyter打开。如直接用文本查看器打开会乱码。

另外各数据集均从作业题中抽取，格式为hw\*ex^.csv，其中\*为作业编号，^为练习题编号
